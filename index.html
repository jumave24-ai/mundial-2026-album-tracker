Acciones rápidas

Para que otras personas vean lo mismo que tú, conecta la app a Google Sheets desde la pestaña Sync. Sin esa conexión, cada celular/PC guarda su propia copia local.

Por sección

Figuritas

La app reconoce códigos como 000, 00, FWC1, COL17, ARG17, USA5 y CC1. También acepta espacios: FWC 1, COL 17, ARG 17, CC 1.

Reportes y PDF

Para PDF, el navegador abrirá el menú de impresión. En Windows puedes elegir “Microsoft Print to PDF”. En celular puedes usar “Guardar como PDF”.

Importar lista oficial o personalizada

La app trae la estructura base 2026: 000 + FWC1-FWC19 + 48 equipos x 20, y también CC1-CC12 de Coca-Cola como bonus. Si tienes una lista con nombres exactos, importa un CSV con columnas: code, section, name, category.

Sync compartido

GitHub Pages publica la app, pero no guarda datos compartidos. Para que todos vean/actualicen lo mismo, esta app se conecta a una Google Sheet usando Google Apps Script.

Links para compartir

Comparte el link de solo lectura con amigos. Para que alguien edite, comparte el mismo link y dale el PIN por separado.

Instalar como app

Cuando la subas a GitHub Pages, abre el link en Chrome/Edge y usa “Instalar esta página como aplicación”. En iPhone usa Safari → Compartir → Agregar a pantalla de inicio.

Configuración

“Regenerar lista faltante” agrega códigos que no existan. No borra lo que ya marcaste. “Borrar datos locales” solo borra este dispositivo, no la Google Sheet.
