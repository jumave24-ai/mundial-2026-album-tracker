Álbum Mundial 2026 - Versión simple v5 sin caché

Esta versión fue creada para evitar el problema de actualización en celular.
No usa service worker ni caché offline. Eso significa que es mucho más fácil de actualizar.

RECOMENDACIÓN PRINCIPAL
Crea un repositorio nuevo en GitHub. No actualices el repositorio viejo.

Nombre sugerido del repositorio:
world-cup-2026-album-simple

Archivos que debes subir a la raíz del repositorio:
- index.html
- manifest.webmanifest
- icon.svg
- google_apps_script_code.gs
- official_stickers_template.csv
- sample_stickers_import.csv
- README_SIMPLE.txt

No subas la carpeta completa. Sube los archivos que están dentro.

Después activa GitHub Pages:
Settings > Pages > Deploy from a branch > main > /root > Save

Abre la app con:
https://TU-USUARIO.github.io/world-cup-2026-album-simple/index.html?v=5

Puedes usar la MISMA Google Sheet y el MISMO Apps Script que ya creaste.
No necesitas crear otra Google Sheet.

Dentro de la app:
1. Ve a Sync compartido.
2. Pega la URL del Apps Script Web App.
3. Escribe tu PIN.
4. Escribe tu nombre.
5. Presiona Guardar conexión.
6. Presiona Cargar desde Google Sheet.

Señal de que es la versión correcta:
Arriba a la derecha debe decir Local · v5 simple o Compartido + edición · v5 simple.
